TeleSint: База Данных

Инструмент для поиска профилей, связей и аналитики внутри мессенджера.

Возможности бота:
  • Поиск по ID, @username или ссылке
  • Поиск по имени профиля (запрос /name)
  • Поиск возможных знакомых по части ID
  • Поиск по пересланному сообщению или контакту
  • Поиск создателей кастомных эмодзи и стикеров
Открыть бота в Telegram
По умолчанию доступно 3 бесплатных запроса. Лимиты можно увеличить внутри бота.
